# ⚽ FIFA World Cup 2026 — Tabela Interativa

Acompanhe a Copa do Mundo FIFA 2026 em um **único arquivo HTML** — sem instalação, sem backend, abre direto no browser.

## 🌐 Demo

👉 **[Abrir agora via GitHub Pages](https://renanhvborges.github.io/copa-2026/copa-2026.html)** ← abre direto no browser, sem instalar nada!

## ✨ Funcionalidades

- **Fase de Grupos** — 12 grupos (A–L), 72 partidas com inputs de placar
- **Classificação automática** — pontos, saldo de gols e critérios de desempate FIFA em tempo real
- **3ºs Lugares** — ranking dos 12 terceiros, destaque dos 8 que avançam
- **Mata-mata** — bracket completo (Round of 32 → Oitavas → Quartas → Semi → Final)
- **Persistência** — resultados salvos automaticamente no `localStorage`
- **Bandeiras reais** — via `flag-icons` (SVG, sem emoji)
- **Identidade visual oficial** — paleta multi-cor da FIFA 26, tema claro, fontes Anton + Bebas Neue

## 🚀 Como usar

1. Baixe o arquivo `copa-2026.html`
2. Abra no seu browser (Chrome, Firefox, Edge, Safari)
3. Preencha os placares e acompanhe a chave se montar automaticamente!

## 🏗️ Stack

| Tecnologia | Uso |
|-----------|-----|
| HTML5 + CSS3 | Estrutura e estilo (tudo embutido) |
